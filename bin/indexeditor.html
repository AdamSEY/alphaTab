alphaTab - Development Showcase

This file shows up how to write notation with alphaTab.
Build: 23.07.2010 v0.2preAlpha

Introduction

To write tabs with alphaTab is really simple.
A song is built up like this: MetaData . Measures It starts with an optional MetaData block which allows you to setup some general song information. The MetaData part is finished with a simple dot. Then a list of measures follows.

MetaData

The metadata can be specified using a kind of tags using the schema: \key value The following example shows which tags are available and how to use them.
\title "Song Title" \subtitle Subtitle \artist Artist \album 'My Album' \words Daniel \music CoderLine \copyright Daniel \tempo 200 \capo 2 \tuning e5 b4 g4 d4 a3 e3 .
Hints:
  • Titles with multiple words require quotes ("doublequotes" or 'singlequotes').
  • Tempos are specified in BPM. By default 120 is set.
  • The Tuning accepts a list of tunings which adds more strings. This allows you to create 4 or 7 instruments. By default the standard 6 string tuning is set.
    Tunings are specified via a note (c, db, d, eb, e, f, gb, g, ab, a, bb, b) and a number specifying the octave.

Measures

Now it gets time to write some measures and notes.
Muliple Measures are seperated by a pipe symbol: |
Hints:
  • Spaces, tabs and newlines (whitespaces) are just ignored within the tabs, but they're often needed to separate and format elements.
. ||||
Single notes per beat can be added in this format: fret.string.duration Rests are written using this format r.duration The duration can be specified as number: 1(full), 2(half), 4(fourth), ... 64 (sixtyfourth).
. 0.6.2 1.5.4 3.4.4 | 5.3.8 5.3.8 5.3.8 5.3.8 r.2
To add multiple notes to one beat, simply use this format ( fret.string fret.string ) .duration
\title "Smoke on the Water" \subtitle "famous riff" . (0.3 0.4).4 (3.3 3.4).4 (5.3 5.4).4 r.8 (0.3 0.4).8 | r.8 (3.3 3.4).8 r.8 (6.3 6.4).8 (5.3 5.4).4 r.4 | (0.3 0.4).4 (3.3 3.4).4 (5.3 5.4).4 r.8 (3.3 3.4).8 | r.8 (0.3 0.4).8

Measure Annotations

You often need to adjust some data on one measure to another. i.E. to change the timesignature or tempo. For this, measures support a similar type of tags like the metadata section at the beginning of a measure. The following sample shows the available tags:
. | \ts 3 4 | \ks Eb | \ro | \rc 2 | \clef bass | \tempo 30

Effects

Dead Notes

To write dead notes, simply replace the fret number with an x.
. x.6.2 x.5.4 x.4.4
